MySQL小表驱动大表

发表于 2020-03-08 更新于 2020-03-10 分类于 MySQL Disqus：

前言

我们查询小表的数据肯定是比查询大表的数据来的快，所以我们可以用查询小表得到的结果集，作为查询条件去查询大表，这样提前的一个结果范围，能够让我们更快的获取数据。
